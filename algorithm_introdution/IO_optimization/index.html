IO Optimization

Content

There are three elements in C++ IO optimization.
  1. cin.tie(0)
  2. ios_base::sync_with_stdio(0)
  3. Use either #define endl '\n' or use '\n' instead
    4.  
Optimization will not work if any of the elements is missed.

		

		

			
Error Example

			

			
In this example, neither #define endl '\n' nor use '\n' instead endl is used, which makes optimization incomplete.
So it would still get a TLE in the testing.

			

		

		

			
Conclusion

			

			
Be careful of every part in IO optimization.
Especially endl.
